GWAS Catalog summary statistics

From 2017 the GWAS Catalog started hosting summary statistics files in addition to curated associations. Additional information can be found on our methods page.

We urge authors of GWAS publications included in the Catalog to contact us if summary statistics are available (FAQ-I2). We can then make these available through the Catalog.

We are currently developing a database of GWAS summary statistics. The REST API for the summary statistics database is now available at www.ebi.ac.uk/gwas/summary-statistics/api.

These data are made available through EMBL-EBI’s standard terms of use.
